# Skill : competitive-insights

## Position concurrentielle unique

**SEO-GEO est le seul outil qui combine :**
1. Audit SEO technique (11 phases, 100pt)
2. Audit GEO natif (llms.txt, AI bots, E-E-A-T)
3. Rapport white-label en **français**
4. Multi-tenant dès V1
5. Check llms.txt comme feature de première classe

**Gap marché confirmé** : aucun concurrent ne fait les 5 simultanément. Semrush $238/mo (SEO + GEO add-on) vs SEO-GEO $149/mo = **37% moins cher** pour les agences franco.

## Concurrents directs (par menace)

### Search Atlas ($99-199/mo) — menace la plus proche
- SEO complet + LLM visibility (QUEST) + white-label
- **Ce qui manque chez eux** : rapport PDF combiné SEO+GEO structuré, langue FR, llms.txt check
- Angle contre-attaque : rapport livrable en FR < 5 min (leur rapport = data dashboard, pas livrable client)

### SE Ranking ($103/mo + add-ons) — menace volume agences
- White-label natif + Agency Pack
- **Ce qui manque** : GEO jeune, pas de rapport unifié SEO+GEO, interface moins raffinée
- Angle : "tout-en-un vs stack à $250+/mo avec add-ons"

### AgencyAnalytics ($79/mo) — opportunité displacement
- **Contexte** : doublement prix par client en octobre 2025 → vague de départs actuelle
- **Ce qui manque** : audit SEO basique (44 checks vs 11 phases), zéro GEO
- Cible : agences déçues cherchant alternative NOW (fenêtre 6-12 mois)
