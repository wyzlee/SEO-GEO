## Contexte produit de référence

**SEO-GEO** est une app SaaS d'audit multilocataire combinant :
- Audit SEO classique (meta, structured data, Core Web Vitals, crawl)
- Audit GEO (llms.txt, AI bots allowlist, signaux E-E-A-T, visibilité dans les LLMs)
- Rapport white-label en français

**Cible V1** : agences SEO, directeurs marketing B2B SaaS  
**Modèle** : agence tool V1 → self-serve V2  
**Positionnement différenciant** : seul outil combinant audit technique SEO + GEO dans un rapport unique, en français, multi-tenant dès V1

## Analyse du paysage marché

### Segmentation du marché 2025-2026

**Segment 1 — SEO classique all-in-one** (Semrush, Ahrefs, SE Ranking, Search Atlas)
- Marché mature, consolidé
- Ajoutent le GEO en add-on / module secondaire
- Prix : $100–500/mo
- Pain : GEO = silo séparé, pas de rapport unifié

**Segment 2 — Audit technique crawl** (Screaming Frog, Sitebulb)
- Outils indispensables mais desktop-first
- Zéro GEO
- Modèle annuel, pas SaaS multi-tenant
- Pain : pas de rapport client automatisé, pas de GEO

**Segment 3 — GEO/AEO monitoring natif** (AthenaHQ, Otterly, Profound, Peec AI)
- Catégorie émergente 2024-2026
- Zéro audit SEO classique
- Hyperfocalisés monitoring/visibility
- Pain : ne remplacent pas l'audit SEO, obligent à multiplier les outils

**Segment 4 — Reporting agences** (AgencyAnalytics, DashThis)
- Outils de reporting, pas d'audit
- Audit SEO très basique
- Pas de GEO
- Pain : manquent la profondeur technique et le GEO

**Segment 5 — Enterprise SEO** (BrightEdge, Conductor)
- $30k–$100k+/an
- Hors de portée des agences SMB
- GEO présent mais opaque

### Gap de marché identifié

**Il n'existe pas aujourd'hui d'outil unique qui :**
1. Combine audit SEO technique complet (300+ checks) + audit GEO natif (llms.txt, AI bots, E-E-A-T, signaux LLM) dans un rapport unique
2. Génère un rapport white-label en français (gap total sur le marché francophone)
3. Est multi-tenant dès V1 (agences gérant N clients depuis une interface)
4. Intègre le check llms.txt comme feature de première classe
5. Est accessible à $99–299/mo (vs. $500+ pour les stacks combinées)

## Implications pour la feuille de route SEO-GEO

### Avantages compétitifs à renforcer

1. **Combinaison SEO + GEO** : c'est le gap principal. Personne ne le fait proprement.
2. **llms.txt checker** : feature de niche à fort potentiel, quasi inexistante ailleurs
3. **Rapport français white-label** : marché francophone (France + DACH francophone + Québec) non adressé
4. **Multi-tenant V1** : Screaming Frog et Sitebulb ne supportent pas le multi-client SaaS
5. **Prix agence** : $99–199/mo pour un outil complet SEO+GEO reste un gap dans le marché

### Risques concurrentiels à surveiller

1. **Semrush One** : intègre SEO + AI Visibility — si Adobe accélère l'intégration post-acquisition, cela réduira le gap
2. **SE Ranking** : au bon rapport qualité/prix, si SE Visible devient plus complet, menace directe sur le segment agence
3. **Otterly.ai / Peec AI** : si l'un d'eux ajoute un audit SEO classique, ils couvrent le gap
4. **Search Atlas** : déjà SEO + LLM visibility + white-label — l'ajout d'un rapport PDF combiné serait très proche
5. **HubSpot AEO** : distribution massive (200k+ clients) — si ils améliorent le produit, absorption du segment B2B SaaS

### Positionnement recommandé pour SEO-GEO V1

> **"L'audit SEO + GEO en un rapport — pour les agences qui veulent garder leurs clients"**

Cible principale : agences SEO francophones gérant 5–50 clients, qui utilisent actuellement Screaming Frog + un outil GEO séparé + AgencyAnalytics pour le reporting (stack à $400–600/mo minimum). SEO-GEO remplace les 3 pour $199/mo.
